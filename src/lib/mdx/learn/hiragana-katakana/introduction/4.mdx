# Learn Hiragana and Katakana

## Lesson 4: Katakana - Main Characters

In this lesson, we'll learn all the **basic syllables of Katakana**, the system used primarily for **foreign words, onomatopoeia, and proper nouns**.
Each character represents a **sound (mora)**, just like Hiragana, and it's essential to recognize its angular and straight forms.
