# Learn Hiragana and Katakana

## Lesson 2: Hiragana - Main Characters

In this lesson, we'll learn all the **basic syllables of Hiragana**, the first Japanese writing system every student should master.

Each character represents a **sound (mora)**, and by combining them, we can form complete words.
